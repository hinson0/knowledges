**UTC（协调世界时，Coordinated Universal Time）** 是当今全球通用的**时间标准**，相当于世界时间的“基准线”。它不是基于某个特定国家或城市的时区，而是由原子钟和国际天文观测共同维持的精确时间。

### 🌍 与东八区的关系

| 概念       | 说明                                   | 例子                      |
| :--------- | :------------------------------------- | :------------------------ |
| **UTC**    | 世界标准时间，无时区偏移。             | 2026-04-24T06:38:40Z      |
| **东八区** | 在 UTC 基础上**加 8 小时**的区域时间。 | 2026-04-24T14:38:40+08:00 |

- **中国标准时间（CST）**：就是东八区时间，比 UTC 快 8 小时。
- **UTC 时间 + 8 小时 = 北京时间**。

### 🕒 为什么需要 UTC？

计算机系统中常以 **UTC 存储** 时间，因为它**无时区歧义**、**全球统一**。需要显示给用户时，再根据用户的时区（如东八区）转换成当地时间。

你之前看到的 `datetime(2026, 4, 24, 6, 38, 40, tzinfo=timezone.utc)` 就是 UTC 时间，对应的**北京时间为同日 14:38:40**。

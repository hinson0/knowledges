### 第一部分：Python + FastAPI + PostgreSQL + Redis 面试题

#### 🗄️ PostgreSQL 与 异步数据库交互

14. **SQLAlchemy 2.0 的异步 ORM (`asyncpg` 驱动) 中，如何处理惰性加载 (`lazy='select'`) 导致的 `MissingGreenlet` 错误？**

### 第二部分：AI Coding（AI 辅助编程）面试题

AI Coding 方向的面试通常分两种：

- **工具使用者层面**：考察如何利用 AI 提升效率、Prompt 技巧、对 IDE 插件的熟练度。
- **平台/算法层面**：考察对 Code LLM 微调、RAG 检索增强、代码向量化的理解。

以下题目兼顾两者。

#### 🤖 AI 编程工具实践 (Copilot / Cursor / Aider)

1. **你是如何编写有效的 Prompt 来生成一个完整的 FastAPI CRUD 模块的？请举出一个具体的 Prompt 模板示例。**
2. **在使用 GitHub Copilot 或 Cursor 时，你遇到 AI 生成的代码存在安全漏洞（如 SQL 注入、硬编码密钥），你会如何修正并利用工具预防？**
3. **Cursor 的 `.cursorrules` 文件有什么作用？如果让你为这个面试项目写一份规则，你会包含哪些指令？**
4. **什么是 Agentic Coding？解释一下 Aider 或 Cursor Composer 是如何实现“多文件编辑”和“终端命令自动执行”的。**
5. **AI 生成的单元测试覆盖率很高但业务逻辑错误，你通常采用什么工作流来验证？（例如：REPL 验证法、隔离测试法）**
6. **如何利用 AI 工具将一个同步阻塞的 Python 脚本重构为 `asyncio` 协程代码？请简述步骤和易错点。**

#### 🔬 代码大模型原理与工程化 (面向高级/架构岗位)

7. **解释一下代码补全模型的 Fill-in-the-Middle (FIM) 训练范式。它是如何让模型理解光标前后上下文的？**
8. **在为企业内部代码库构建私有化 AI 编码助手时，RAG（检索增强生成）与 Fine-Tuning（微调）在成本、效果、更新频率上的主要区别是什么？**
9. **如何对代码片段进行向量化（Embedding）？代码的 AST（抽象语法树）特征会比纯文本特征更具优势吗？**
10. **假设你要让 AI 精准回答关于公司内部 `ORM` 封装库的问题，你会如何构建一个代码知识库索引流程？**
11. **如何评估一个 AI 编程模型的输出质量？除了 Human Eval 分数，你还会关注哪些具体的业务指标（如：Acceptance Rate, Character Saved）？**
12. **谈谈你对当前 AI Coding 工具局限性的看法：在处理复杂重构（如跨大项目的重命名）或非主流语言（如 Elixir）时，为什么它们表现会下降？**

#### 🛠️ 动手设计题 (System Design for AI Coding)

13. **设计题：** 给你 100 万行 Python 单体遗留代码仓库，需要构建一个“代码智能问答机器人”，要求回答能准确指到具体文件和行号。请简述你的技术选型（向量数据库、分块策略、检索算法）。
14. **安全题：** 公司的法务部门担心开发者将核心 C++ 算法代码粘贴到 ChatGPT 网页端。作为技术负责人，你会采取哪三种措施在不降低开发效率的前提下防止数据泄露？

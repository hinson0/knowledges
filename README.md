# 🧠 Knowledges

> 学习笔记与技术实践的知识仓库 — 由 [`learning`](https://github.com/hinson0/learning) 迁至

📚 **Knowledges** 是一个聚焦后端开发、云原生、移动端与前端技术的 **学习笔记仓库**，以项目驱动、模块化组织的思路记录各方向的技术要点与最佳实践。

---

## 📂 目录结构

| 目录          | 主题           | 说明                             |
| ------------- | -------------- | -------------------------------- |
| `ai-native/`  | AI 原生应用    | 大模型集成、AI 服务化开发        |
| `android/`    | Android 开发   | Kotlin/Java，Android 体系        |
| `cc/`         | C/C++          | 基础语法与实践                   |
| `docker/`     | Docker         | 容器化、镜像构建、Docker Compose |
| `ecc/`        | 椭圆曲线密码学 | 密码学原理与工程实现             |
| `fastapi/`    | FastAPI        | 现代 Python Web 框架实践         |
| `image/`      | 图片资源       | 项目相关静态资源                 |
| `just/`       | Just           | 命令运行器 Justfile 的使用       |
| `monorepo/`   | 单体仓库       | 多项目管理与工具链               |
| `nginx/`      | Nginx          | 反向代理、负载均衡、配置最佳实践 |
| `pnpm/`       | pnpm           | 快速、节省磁盘的包管理器         |
| `postgres/`   | PostgreSQL     | 关系型数据库原理与调优           |
| `pydantic/`   | Pydantic       | 数据验证与配置管理               |
| `python/`     | Python         | 核心语法、标准库与进阶技巧       |
| `sqlalchemy/` | SQLAlchemy     | ORM 设计与使用                   |
| `sqlite3/`    | SQLite3        | 轻量级数据库实践                 |
| `vue3/`       | Vue 3          | 组合式 API、响应式原理与生态     |

---

## 🚀 使用方式

```bash
# 克隆仓库
git clone https://github.com/hinson0/knowledges.git
cd knowledges

# 浏览某个主题
cd docker/
ls
```

各子目录内主要为按具体话题分散存放的 Markdown 笔记文件，少数目录下含有进一步的子目录或代码文件。
